# Suno Helper

Browser-based pitch correction tool, inspired by Melodyne. Record, import, visualize, and correct pitch — all in the browser with no backend.

## Features

- Record audio or import files (WAV, MP3, FLAC, OGG, M4A, etc.)
- Real-time pitch detection using McLeod Pitch Method (pitchy)
- Visual pitch editor with draggable note blobs
- Auto-correct to nearest semitone or manual adjustment
- Export corrected audio as WAV
- Undo/redo support

## Tech Stack

React 19, TypeScript, Vite, Zustand, Tailwind CSS, Web Audio API, Canvas

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
